📖 About the Project

The To-Do List App is a beginner-friendly web project that demonstrates how to build an interactive application using Vanilla JavaScript.

It allows users to:

Manage daily tasks

Track completed tasks

Store tasks permanently using the browser's local storage

This project is great for learning DOM manipulation, event handling, and local storage.

✨ Features

✔ Add new tasks
✔ Mark tasks as completed
✔ Delete tasks
✔ Persistent data using Local Storage
✔ Clean and simple UI
✔ Beginner-friendly project

🛠 Technologies Used
Technology	Purpose
HTML5	Structure of the webpage
CSS3	Styling and layout
JavaScript	Application logic
LocalStorage API	Saving tasks in browser

⚙️ How It Works

1️⃣ Enter a task in the input field
2️⃣ Click the Add button
3️⃣ The task will appear in the list

You can then:

✔ Mark the task as completed
🗑 Delete the task

All tasks are stored in Local Storage, so they remain after refreshing the page.

🚀 Getting Started

Follow these steps to run the project locally.

1️⃣ Clone the repository
git clone https://github.com/op0710/todo-list-app.git
2️⃣ Open the project folder
3️⃣ Run the project

Open index.html in your browser.
